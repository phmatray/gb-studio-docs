# Erreur

Oh non, vous avez rencontré une erreur dans GB Studio...

À ce stade, avant de faire quoi que ce soit, vous devriez faire une sauvegarde de votre dossier de projet au cas où quelque chose se passerait.

Une fois que vous avez fait une sauvegarde, vous pourriez toujours être en mesure de sauvegarder votre projet en utilisant `Fichier / Enregistrer` dans le menu. Si cela échoue, revenez à la sauvegarde.

## Signaler des problèmes

Si vous êtes tombé sur une erreur dans GB Studio, nous apprécierions grandement que vous signaliez un problème à l'adresse :
https://github.com/chrismaltby/gb-studio/issues

## Obtenir de l'aide

Le meilleur endroit pour obtenir de l'aide lorsque GB Studio ne fonctionne pas est soit le [canal Discord](https://discord.gg/bxerKnc) soit sur Reddit à [/r/GBStudio](https://www.reddit.com/r/gbstudio).
